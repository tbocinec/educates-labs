## Workshop Steps

### **Step 1:** Launch All Services
- Start 3 Grafanas, InfluxDB, and ClickHouse with Docker Compose
- Verify all services are running
- Access each Grafana instance

### **Step 2:** Setup Data Sources
- Configure **Grafana1** with InfluxDB data source
- Configure **Grafana2** with ClickHouse data source
- Create basic dashboards in each Grafana

### **Step 3:** Configure Federation
- Add **Grafana1** as data source in **Grafana3**
- Add **Grafana2** as data source in **Grafana3**
- Test federation connectivity

### **Step 4:** Create Federated Dashboards
- Build unified dashboard in **Grafana3**
- Combine sensor data and business metrics
- Create cross-system visualizations

## Why Federation?

**Benefits:**
- âœ… **Centralized view** - Single dashboard for all data
- âœ… **Data stays at source** - No data duplication
- âœ… **Team autonomy** - Each team manages their own Grafana
- âœ… **Scalable architecture** - Easy to add new data sources
- âœ… **Cross-system insights** - Correlate different data types

**Use Cases:**
- **Multi-team environments** - Each team has specialized Grafana
- **Different data types** - Time series + SQL analytics
- **Legacy system integration** - Connect old and new systems
- **Geographic distribution** - Federate across locations
