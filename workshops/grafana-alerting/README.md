# Grafana Alerting Workshop

A hands-on workshop for learning the basics of alerting in Grafana with ClickHouse using Grafana Unified Alerting.

## Overview

This workshop teaches you the fundamentals of Grafana alerting. You'll set up a simple API monitoring scenario with ClickHouse, create a threshold alert, configure webhook notifications, and trigger the alert to see the complete workflow.

## What You'll Learn

- Set up ClickHouse with API response time metrics
- Configure webhook notifications
- Create a threshold-based alert rule with Grafana Unified Alerting
