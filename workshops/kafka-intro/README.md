# Kafka Intro Workshop (Educates)

Tento workshop slúži ako jednoduchá ukážka Apache Kafka bežiacej v rámci Educates prostredia.

Účastník si v niekoľkých moduloch vyskúša:

1. Prácu s Terminal a Editor aplikáciami v Educates.
2. Vytvorenie `docker-compose.yml` so Zookeeper + Kafka kontajnermi.
3. Spustenie Kafka stacku pomocou `docker compose`.
4. Vytvorenie topicu a posielanie/čítanie správ pomocou Kafka CLI.

Workshop je zámerne jednoduchý a slúži ako základ, na ktorý sa dajú neskôr pridávať ďalšie moduly (Kafka UI, Java klient, viac brokerov, atď.).
